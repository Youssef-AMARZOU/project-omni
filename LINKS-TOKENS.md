## 🐙 GitHub Token

### Lien direct
**https://github.com/settings/tokens**

### Étapes
1. Cliquez le lien ci-dessus
2. Cliquez **"Generate new token (classic)"**
3. Remplissez :
   - **Note** : `Project OMNI`
   - **Expiration** : `No expiration` (ou 90 jours)
   - ✅ Cochez **"repo"** (tout le bloc)
4. Cliquez **"Generate token"** en bas
5. **Copiez immédiatement** le token (commence par `ghp_`)

## 🤗 HuggingFace Token

### Lien direct
**https://huggingface.co/settings/tokens**

### Étapes
1. Cliquez le lien ci-dessus
2. Cliquez **"New token"** (bouton orange en haut à droite)
3. Remplissez :
   - **Token name** : `Project OMNI`
   - **Role** : Sélectionnez **"Write"** ⚠️ (pas "Read")
4. Cliquez **"Create token"**
5. **Copiez immédiatement** le token (commence par `hf_`)

2. Cliquez **"Create API token"**
